AWS - Week 1

Some fundamental services in AWS i explored:

- EC2 - Elastic Cloud Compute: It is a virtual machine available in cloud. We can pick up the OS, RAM and CPU. It is like renting a server.

- S3 - Simple Storage Service: It is a object storage service. Allows to store any object with a url. It is like Google Drive but accessed via API

- Lambda : A function can be uploaded without any server. The function will be executed only on request. This is very useful for lightweight automation tasks.

- IAM - Identity and Access Management - This service allows to maintain role permissions for the AWS account

- RDS - Relational Database Service - Provides managed DB service along with backups, replicas, patching etc.
